Defining the following tags

Article

Article tag is to identify self contained content. For instance, a word and its definition

Details and Summary

The details tag specifies details you can open or close by clicking on the word

The summary tag specifies a heading for the details tag

Time and datetime

The time tag specifies a specifc time 09:00

The datetime tag can make a date saveable on the user's calendar. Also used by search engines to optimize search results
